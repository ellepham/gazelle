## 3. AI-Specific Design Principles

_If your product uses AI, what principles guide AI-human interaction?_

These universal principles apply to most AI-powered products:

| Principle            | Guideline                                                               |
| -------------------- | ----------------------------------------------------------------------- |
| Transparency         | Show users what the AI did and why. No black boxes.                     |
| User control         | AI suggests, user decides. Never auto-apply without consent.            |
| Graceful degradation | When AI is wrong, make it easy to correct.                              |
| Honest confidence    | Show uncertainty. "Likely X" is better than presenting a guess as fact. |
| Reversibility        | Every AI action should be undoable.                                     |
